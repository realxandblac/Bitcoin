# Bitcoin Price Tracker

## Overview
The Bitcoin Price Tracker is a project designed to provide real-time monitoring of Bitcoin prices across various exchanges. This tool helps users to keep track of price movements, analyze trends, and make informed decisions regarding their Bitcoin investments.

## Features
- **Real-time Price Updates**: Get the latest Bitcoin prices fetched from multiple exchanges.
- **Historical Data Analysis**: Access to historical price data to analyze trends over time.
- **Alerts and Notifications**: Users can set price alerts to get notified when the price hits specific thresholds.
- **User-friendly Interface**: The application comes with an intuitive interface for easy navigation and use.

## Technologies Used
- Programming Language: Python
- Libraries: requests, pandas, matplotlib
- Database: SQLite (for storing historical data)
- Frontend: Flask for web interface

## Installation
To get started with the Bitcoin Price Tracker:
1. Clone the repository:
   ```bash
   git clone https://github.com/realxandblac/Bitcoin.git
   cd Bitcoin
   ```
2. Install the required libraries:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   python app.py
   ```
4. Access the application at `http://localhost:5000`.
